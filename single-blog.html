Generation of Challan for not wearing helmet in two wheelers.

This Flask-based web application is tailored to detect whether motorcycle riders are wearing helmets and to recognize their license plates. Utilizing machine learning algorithms, it automatically generates traffic violation tickets for riders found without helmets. By incorporating Flask, a Python web framework, the application provides a user-friendly interface for managing violations and issuing tickets. This system enhances safety enforcement measures on the road while promoting compliance with helmet-wearing regulations. GitHub Link
